Detailed Description

Bloom filter policy that uses a bloom filter with approximately the specified number of bits per key.

Note: if you are using a custom comparator that ignores some parts of the keys being compared, you must not use this

BloomFilter

and must provide your own FilterPolicy that also ignores the corresponding parts of the keys. For example, if the comparator ignores trailing spaces, it would be incorrect to use a FilterPolicy (like

BloomFilter

) that does not ignore trailing spaces in keys.
