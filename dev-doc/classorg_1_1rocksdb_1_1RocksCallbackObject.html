Detailed Description

RocksCallbackObject is similar to RocksObject but varies in its construction as it is designed for Java objects which have functions which are called from C++ via JNI.

RocksCallbackObject is the base-class any RocksDB classes that acts as a callback from some underlying underlying native C++

rocksdb

object.

The use of

RocksObject

should always be preferred over RocksCallbackObject if callbacks are not required.
