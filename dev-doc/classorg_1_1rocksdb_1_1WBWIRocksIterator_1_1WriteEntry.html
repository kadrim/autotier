Detailed Description

Represents an entry returned by org.rocksdb.WBWIRocksIterator#entry()

It is worth noting that a WriteEntry with the type org.rocksdb.WBWIRocksIterator.WriteType#DELETE or org.rocksdb.WBWIRocksIterator.WriteType#LOG will not have a value.
